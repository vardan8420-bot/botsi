# 🤖 Botsi - AI Super Bot

Мощный AI Telegram бот с поддержкой армянского языка.

## Возможности

### Часть 1 - Ядро ✅
- 🇦🇲 Армянский язык (հայերեն + транслит)
- 🧠 AI (GPT-4o-mini + GPT-4o)
- 💾 Память разговора
- ⚡ Кеширование

### Часть 2 - Контент (в разработке)
- 🎤 Голосовое управление
- 🔊 Голосовые ответы
- 🎬 Создание видео
- 📱 Автопостинг

### Часть 3 - Бизнес (в разработке)
- 📊 Менеджер маркетплейсов
- 📦 Управление складом
- 📈 ML аналитика
- 🔍 Мониторинг конкурентов

### Часть 4 - Разработка (в разработке)
- 💻 GitHub интеграция
- 🔍 Анализ кода
- 🤖 Авто-разработчик
- 🎨 UI/UX дизайн

## Деплой на Railway

1. Подключить GitHub репо
2. Добавить переменные: `BOT_TOKEN`, `OPENAI_API_KEY`
3. Автоматический деплой

## Использование

Просто напишите боту - не нужен /start!

Примеры:
- "Voghchuyn" → "Բարև! 😊"
- "Как дела?" → ответ на русском
- "Create a landing page" → создаёт лендинг

## Команды

- `/help` - показать справку
- `/language hy|ru|en` - сменить язык
- `/stats` - статистика пользователя
- `/reset` - очистить историю разговора

## Установка локально

```bash
# Клонировать репозиторий
git clone https://github.com/vardan8420-bot/botsi.git
cd botsi

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env
# Заполнить BOT_TOKEN и OPENAI_API_KEY

# Запустить бота
python bot.py
```
